## Overview
This repository contains practical Python projects and code samples for a Computer Vision course. Each chapter demonstrates key concepts and techniques in computer vision using OpenCV and NumPy. The projects are designed to help students understand and implement real-world computer vision tasks.

## Contents
- **chap01.py – chap22.py:**
  - Each file corresponds to a chapter or topic in the course, covering:
    - Image and video processing
    - Edge detection
    - Color space transformations
    - Object and face detection
    - Feature extraction
    - Camera operations
    - And more
- **images/**
  - Contains sample images and outputs used in the projects.

## How to Run
1. Make sure you have Python 3.x installed.
2. Install required packages:
   ```bash
   pip install opencv-python numpy
   ```
3. Run any chapter script:
   ```bash
   python chapXX.py
   ```
   Replace `XX` with the chapter number you want to run.

## Requirements
- Python 3.x
- OpenCV (cv2)
- NumPy
